# nanoavionics

## Monitor

Script file is located in scripts folder by name monitor.exe , execution has to be done in cmd or windows terminal. <br>

|Flag|Explanation|
|----|-----------|
|-f  |filename   |
|-d  |directory  |
|-q  |query      |
|-m  |monitor    |
|-r  |reader     |
|-s  |chunk size |

Sample commands:
`monitor.exe`
